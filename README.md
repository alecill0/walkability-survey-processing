# walkability-survey-processing
This project processes pedestrian survey responses and transforms qualitative answers into quantitative variables for clustering and walkability analysis


This project focuses on the processing and calibration of pedestrian survey data for walkability analysis.

The workflow transforms qualitative survey responses into quantitative variables suitable for clustering and statistical analysis.

## Main Features
- Data cleaning and preprocessing
- Encoding of qualitative responses
- Survey calibration and normalization
- Preparation of datasets for clustering analysis

## Tools Used
- Python
- pandas
- numpy
- scikit-learn

## Project Status
Completed academic project used for walkability and pedestrian clustering analysis.
